📝 Pasos para cambiar el color:

  1. Abre tu archivo css/styles.css
  2. Busca la sección :root { (línea 2-9)
  3. Reemplaza estos valores:
    • --accent-color: #TU_COLOR_PRINCIPAL;
    • --accent-light: #TU_COLOR_CLARO;
    • --accent-dark: #TU_COLOR_OSCURO;
  4. Guarda el archivo (Ctrl + S)
  5. Recarga tu navegador (F5)
  6. ¡Listo! Tu portfolio ahora tiene nuevos colores
